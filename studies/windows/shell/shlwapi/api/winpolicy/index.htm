Windows Policies

In the SHLOBJ.H file from the Software Development Kit (SDK) for Windows Vista, Microsoft says that the SHRestricted function “has been deprecated in favor of SHWindowsPolicy”. The former was documented only grudgingly by Microsoft as part of settling a law suit. Its supposed, or planned, replacement is not documented at all.

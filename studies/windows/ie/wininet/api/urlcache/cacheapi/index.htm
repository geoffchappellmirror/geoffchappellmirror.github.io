The URL Cache API

Internet Explorer has long provided for caching just about anything that might be needed to view a web page. WININET exports a substantial set of relevant functions. In the list that follows, functions that seem not to be documented are highlighted yellow. Some of the others have become documented only in the last few years. Note however that even the undocumented functions are at least declared in an SDK header file (WININETI.H).

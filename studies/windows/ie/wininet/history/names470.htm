WININET 4.70 Named Exports

Version 4.70 of WININET.DLL was the first to be distributed with any version of Windows. In the crude terms of function numbers, this version roughly doubled the API. Many of the additions, however, had an earlier existence as exports from URLCACHE.DLL, which is known only from the original Internet Explorer 3.0.

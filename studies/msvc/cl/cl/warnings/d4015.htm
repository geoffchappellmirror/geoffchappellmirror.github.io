CL Command-Line Warning D4015

Message Text 

invalid assembly listing configuration 'option'

This warning message is written to whatever file or device was given to CL as standard error. It is therefore not subject to simple redirection from the Command Prompt, e.g., for capture to a file.

Circumstances

The option is necessarily /FA. The argument, which is reproduced in the message text, is expected to be any number, including none, of the following case-sensitive characters: a, b, c and s.

API Sets Added For Windows 10.0

The ApiSetSchema.dll supplied with Windows 10 defines 624 (x86) or 626 (x64) API Sets. The very large table that follows lists the added API sets. There are very many more (306) than might at first be expected just from the increase over sets defined for version 6.3. This is because version 10.0 discontinues many sets that are in some sense superseded.

LINK /FE

This option names the front-end compiler.

Syntax

/fe:dllname

The required dllname argument consists of everything after the colon.

Behaviour

The dllname argument is expected to name a DLL that exports a function named IDLMerge. Interpretation of dllname, e.g., to find the DLL if no drive or path is specified, is up to LoadLibrary. With no /fe option having provided a dllname, LINK uses “c1xx.dll”.

The option is acted upon only if LINK must merge IDL content. It is a fatal error (LNK1171) if the DLL does not load. It is a fatal error (LNK1173) if the DLL does not export the expected function. What that function does with the IDL content and what LINK does in response is beyond the scope of this note.

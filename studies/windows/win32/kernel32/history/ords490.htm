Ordinal-Only Exports Added For KERNEL32 4.90

Version 4.90, for Windows Me, added just one KERNEL32 function that is exported only by ordinal. As with all ordinal-only exports from KERNEL32, it is not documented and it is specific to the Windows that runs on DOS.

Ordinal Function
123 GetModuleNameFromProc

CL /Zc

Syntax

/Zc:keyword

where the required keyword is either forScope or wchar_t.

The supported keywords can be given in one /Zc option by using the syntax of combined options.

Relations

This option has no syntactic relations with other options: it does not override any; it is not incompatible with any; it has no prerequisites.

This option passes to the front-end C++ compiler (C1XX) only.

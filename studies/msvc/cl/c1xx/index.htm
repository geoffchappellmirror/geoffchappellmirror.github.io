The Front-End C++ Compiler (C1XX)

The loading and execution of C1XX.DLL is ordinarily coordinated through CL.EXE and is typically not something the programmer is very much aware of. However, it is C1XX that actually compiles C++ source code to an intermediate language suitable for code generation.

Kernel Exports Added for Windows Vista SP2

This page lists the functions that are first exported by the Windows kernel in the version 6.0 from Windows Vista SP2. There is just the one. it is still exported as of version 10.0 but is not yet documented. Microsoft has, however, published a C-language declaration, just the once, plausibly by mistake, in the 1511 edition of the Enterprise WDK for Windows 10.

  • IoGetSymLinkSupportInformation

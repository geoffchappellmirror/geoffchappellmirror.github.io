Internet Explorer Extensions

URLMON version 7.0 introduces more precise control of browser extensions. Along with the new functionality come new functions. All are exported only by ordinal. None were already documented in the January 2007 edition of the Windows SDK, i.e., in time for the release of Windows Vista.

  • CoInternetApproveExtension
  • CoInternetCreateExtension
  • CoInternetExtensionAllowed
  • CoInternetExtensionCollectStats
  • CoInternetExtensionNeedsApproval
  • CoInternetIsExtensionsOff
  • CoInternetSetExtensionsOff
  • IEIsProtectedModeIUriInternal
  • IsProtectedModeIUri
  • IsProtectedModeURL
  • SkipBrokerCheckForURL

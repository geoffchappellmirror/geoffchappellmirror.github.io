SYSTEM_SPECIAL_POOL_INFORMATION

The SYSTEM_SPECIAL_POOL_INFORMATION structure is produced as output or expected as input by the functions

  • ZwQuerySystemInformation or NtQuerySystemInformation;
  • ZwSetSystemInformation or NtSetSystemInformation;

respectively, when given the information class SystemSpecialPoolInformation (0x57).

Documentation Status

The SYSTEM_SPECIAL_POOL_INFORMATION structure is not documented.

Layout

The SYSTEM_SPECIAL_POOL_INFORMATION is 0x08 bytes in both 32-bit and 64-bit Windows.

Offset Definition
0x00 
ULONG PoolTag;
0x04 
ULONG Flags;

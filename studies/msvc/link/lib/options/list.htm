LIB /LIST

Syntax

/list[:listfile]

The optional listfile argument consists of everything after the colon.

Behaviour

The /list option puts LIB into the mode in which it lists the members of whichever libraries are among the input files. Note however that /def has precedence.

The listing is written to the named listfile, else to standard output.

API-MS-Win-Core-DelayLoad-L1-1-0

The one function in the API-MS-Win-Core-DelayLoad-L1-1-0 set is an undocumented export from KERNEL32 in version 5.0 and higher:

  • DelayLoadFailureHook

The Windows 7 schema redirects this API Set to KERNEL32.

Registry Strings

SHLWAPI version 5.00 introduced a function for loading a string from the registry and interpreting it in a particular format. Version 6.00 added a function which does just the interpretation.

  • SHLoadIndirectString (6.00)
  • SHLoadRegUIString

The original function is exported only by ordinal and seems never to have got documented.

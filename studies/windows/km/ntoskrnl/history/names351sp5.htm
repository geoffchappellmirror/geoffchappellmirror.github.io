Kernel Exports Added for Version 3.51 SP5

This page used to list all the exports that were added to the Windows kernel from version 3.51 after its original release, i.e., in service packs. There are only four. Only SP5 had yet been obtained for inspection. One of the four turns out to have been added as early as SP2. All four are now listed among the Kernel Exports Added for Version 3.51.

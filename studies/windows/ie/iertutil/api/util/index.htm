Utilities

IERTUTIL exports nearly a dozen functions for general utility. Most are original, for version 7.0, but two were added for version 8.0:

  • GetCurrentUserSid (8.0)
  • GetProcessIntegrityLevel
  • GetProcessSid
  • GetUserSid
  • IECheckIntegrityFile (8.0)
  • IEIsLowIntegrityFile
  • IERunsAsDifferentUser
  • IERunsOnDefaultDesktop
  • IsMICLowProcess
  • SetFileHandleIntegrityLevel
  • SetKernelHandleIntegrityLevel
  • SetKernelHandleToLowIL
  • SetRegistryKeyIntegrityLevel

IERTUTIL 8.0 Ordinal-Only Exports

Version 8.0 adds several hundred functions that are exported only by ordinal.

The column of Remarks and the usual colour-coding scheme for documentation status await my receipt (on disc from an MSDN subscription) of a definitive SDK for Windows 7 to adopt as a formal reference edition of Microsoft’s documentation. Meanwhile, these functions that are exported only by ordinal are assumed to be undocumented, as is Microsoft’s typical practice.

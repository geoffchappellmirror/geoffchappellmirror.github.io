Supporting the About Internet Explorer Dialog

From as long ago as version 4.71, SHLWAPI.DLL exports a function that seems to exist solely to support the dialog box that is reached by clicking About Internet Explorer in the Help menu of Internet Explorer. This function is exported by ordinal only and is not documented. Its functionality is sufficiently separate from the rest of SHLWAPI that it has its own source file.

Kernel Exports Added for Windows 2000 SP3

The table below lists the five exports that were first added to the Windows kernel for the version 5.0 from Windows 2000 SP3. All are still exported as of version 10.0. Of the three that were not documented immediately, two have so far been documented only to say that they’re reserved.

Function Remarks
IoIsFileOriginRemote documentation requires Windows XP and higher
IoSetFileOrigin documentation requires Windows XP and higher
RtlPrefetchMemoryNonTemporal undocumented until 2006-2007;
documentation requires Windows Server 2003 and higher
SeAuditHardLinkCreation undocumented until 2000-2006 (IFS);
declaration requires Windows XP and higher
SeAuditingHardLinkEvents undocumented until 2000-2006 (IFS)

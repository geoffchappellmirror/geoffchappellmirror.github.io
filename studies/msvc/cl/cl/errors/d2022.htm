CL Command-Line Error D2022

Message Text 

cannot open 'filename'

Circumstances

The named file is a command file (which the product documentation sometimes calls a compiler response file). Error D2022 means that CL failed at opening the file for reading.

Documentation Status

The product documentation explains this error in terms of something called a “message file” that “contains a brief summary of compiler command-line syntax and options.” Perhaps this explanation was true for this error once upon a time.

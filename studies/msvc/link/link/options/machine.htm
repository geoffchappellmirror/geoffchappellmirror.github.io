LINK /MACHINE

This option sets the machine type for the executable.

Syntax

/machine:machine

The required argument is a simple keyword. Permitted values are AM33, AMD64, ARM, I386, IA64, IX86, M32R, MIPS, MIPS16, MIPSFPU, MIPSFPU16, MIPSR41XX, PPC, PPCFP, SH3, SH3DSP, SH4, SH5, THUMB, TRICORE, X86 and X86-64. Anything else causes a warning (LNK4012).

Several of the keywords are aliases: I386 and IX86 for X86, and X86-64 for AMD64.

The /machine option is for the command line only. It is not valid as a directive in an object file.

Documentation Status

The text for warning LNK4012 omits AMD64. So too does the help text produced by the link /? command. The Visual C++ documentation in the January 2004 edition of the MSDN Library edition on CD omits AMD64 from the description of /machine but mentions it in passing for /opt.

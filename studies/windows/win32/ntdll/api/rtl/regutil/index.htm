Registry Utilities

It won’t surprise anyone that the Run-Time Library (RTL) in NTDLL exports functions to ease the work of accessing information that is kept in the registry. Many have been exported for all the history of Windows, i.e., since version 3.10.

  • RtlCheckRegistryKey;
  • RtlCreateRegistryKey;
  • RtlDeleteRegistryValue;
  • RtlExpandEnvironmentStrings (3.10, then 6.0 and higher);
  • RtlExpandEnvironmentStrings_U;
  • RtlFormatCurrentUserKeyPath (3.51 and higher);
  • RtlGetNtGlobalFlags;
  • RtlOpenCurrentUser;
  • RtlQueryDynamicTimeZoneInformation (6.0 and higher);
  • RtlQueryRegistryValues;
  • RtlQueryRegistryValuesEx (6.2 and higher);
  • RtlQueryTimeZoneInformation;
  • RtlSetDynamicTimeZoneInformation (6.0 and higher);
  • RtlSetTimeZoneInformation;
  • RtlWriteRegistryValue;
  • RtlpCheckDynamicTimeZoneInformation (6.0 from Windows Vista SP1, and higher).

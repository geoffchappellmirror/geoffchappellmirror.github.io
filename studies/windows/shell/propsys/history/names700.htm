PROPSYS 7.0 Named Exports

The PROPSYS version 7.0 from Windows Vista SP2 exports the same functions as the version 6.0 from Windows Vista SP1. However, the version 7.0 from Windows 7 does have new functions that are exported by name. They are all documented.

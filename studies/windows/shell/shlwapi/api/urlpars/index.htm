URL Parsing

From the very beginning, SHLWAPI has included among its exported functions as least some that work with URLs. Almost all date from version 4.71 and have long been documented, though only as being available from version 5.0 onwards.

  • HashData
  • ParseURL (4.70)
  • PathCreateFromUrl
  • UrlApplyScheme
  • UrlCanonicalize
  • UrlCombine
  • UrlCompare
  • UrlCrackW (5.50)
  • UrlCreateFromPath
  • UrlEscape
  • UrlFixupW (5.0 from Windows 2000)
  • UrlGetLocation
  • UrlGetPart
  • UrlHash
  • UrlIs
  • UrlIsNoHistory
  • UrlIsOpaque
  • UrlUnescape

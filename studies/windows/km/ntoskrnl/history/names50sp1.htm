Kernel Exports Added for Windows 2000 SP1

The table below lists all the exports that were added to the Windows kernel for the version 5.0 from Windows 2000 SP1. All are still exported in version 10.0.

None of these functions are documented in the Windows Vista Windows Driver Kit (WDK), nor even declared in any WDK header file. In the usual practice of these notes, they are therefore highlighted yellow.

  • RtlTraceDatabaseAdd
  • RtlTraceDatabaseCreate
  • RtlTraceDatabaseDestroy
  • RtlTraceDatabaseEnumerate
  • RtlTraceDatabaseFind
  • RtlTraceDatabaseLock
  • RtlTraceDatabaseUnlock
  • RtlTraceDatabaseValidate

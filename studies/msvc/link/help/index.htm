The Helper

The Helper is executed by running the command link /help. It displays the following text: 

LINK /Link for help on Linker
LINK /Lib for help on Library Manager
LINK /Dump for help on Dumper
LINK /Edit for help on Editor
LINK /Help for help on Helper

(where the LINK at the beginning of each line is an upper-case version of whatever filename the program has when run). This is all the Helper exists to do. Additional command-line arguments are irrelevant and are ignored without complaint.

In version 7.00.9466, and presumably in at least some others, a simple programming error produces an Internal error during SetupPhase instead of the expected text—yes, just from running link /help.

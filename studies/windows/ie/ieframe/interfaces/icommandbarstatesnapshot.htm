ICommandBarStateSnapshot

IID_ICommandBarStateSnapshot {8AD2BB74-1382-4C5E-99BA-6485A56D9D54}

The ICommandBarStateSnapshot interface has just the one method (after the IUnknown methods):

Offset Method
0x0C get_Data

with prototype: 

HRESULT get_Data (UINT, PBYTE);

Availability

The ICommandBarStateSnapshot interface is implemented in IEFRAME version 7.0 only.

This interface is not documented by Microsoft as late as the January 2007 edition of the Software Development Kit (SDK) for Windows Vista.

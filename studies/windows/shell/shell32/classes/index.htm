SHELL32 Classes

SHELL32 exposes numerous classes through its DllGetClassObject function. Given that they are properly listed in the registry, these SHELL32 classes are creatable COM objects. The following are the CLSIDs that are supported in the SHELL32 version 6.00 from Windows Vista. The many that are highlighted yellow appear to be undocumented, specifically in the sense that no CLSID symbol is declared for them in any header file from the Windows Vista SDK dated January 2007.

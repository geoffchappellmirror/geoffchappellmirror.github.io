MSHTML Classes: CBlockElement

An instance of the CBlockElement class is created for each ADDRESS, BLOCKQUOTE, CENTER, LISTING, PLAINTEXT, PRE or XMP tag.

Identifiers

coclass: HTMLBlockElement {3050F281-98B5-11CF-BB82-00AA00BDCE0B}
dispinterface: DispHTMLBlockElement {3050F506-98B5-11CF-BB82-00AA00BDCE0B}

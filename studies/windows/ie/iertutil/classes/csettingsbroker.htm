CSettingsBroker

The CSettingsBroker class is at present (version 8.0) the only class that IERTUTIL exposes through its exported DllGetClassObject function. Instances can be created using such well-documented COM functions as CoCreateInstance, given the CLSID:

CLSID_CSettingsBroker {C6CC0D21-895D-49CC-98F1-D208CD71E047}

The class implements just the one interface (other than IUnknown):

  • ISettingsBroker

CL /d1_5

This option adds an option to the command line for the in-between C/C++ compiler.

Syntax

/d1_5option

The required argument consists of whatever follows immediately within the same command-line token, else is the whole of the next command-line token.

Behaviour

This option generates -option for the in-between C/C++ compiler, which is ordinarily not defined (but which can be enabled and named through the /B1_5 option).

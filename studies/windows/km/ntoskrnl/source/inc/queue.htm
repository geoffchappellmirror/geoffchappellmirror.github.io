QUEUE.H

The public symbol file NTKRPAMP.PDB for the original release of Windows 10 tells that the kernel is built with the QUEUE.H header at

d:\th\minkernel\ntos\io\pnpmgr

and draws from it the following type definition:

Line Number Type
36 struct _PNP_DEVICE_COMPLETION_QUEUE

The header QUEUE.H is not known in any Device Driver Kit (DDK) or Windows Driver Kit (WDK).

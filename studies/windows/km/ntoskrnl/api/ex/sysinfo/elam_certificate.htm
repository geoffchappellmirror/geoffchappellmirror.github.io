SYSTEM_ELAM_CERTIFICATE_INFORMATION

The SYSTEM_ELAM_CERTIFICATE_INFORMATION structure is expected as input to ZwSetSystemInformation or NtSetSystemInformation when given the information class SystemElamCertificateInformation (0x98).

Documentation Status

The SYSTEM_ELAM_CERTIFICATE_INFORMATION structure is not documented.

Layout

The SYSTEM_ELAM_CERTIFICATE_INFORMATION is 0x04 or 0x08 bytes in 32-bit and 64-bit Windows, respectively.

Offset (x86) Offset (x64) Definition
0x00 0x00 
PVOID ElamDriverFile;

ITridentService3

IID_ITridentService3 {620CC4C4-C9C6-11DA-95AF-00E08161165F}

The ITridentService3 interface has just the one method (after the IUnknown methods):

Offset Method
0x0C FireBeforeNavigate3

with prototype: 

HRESULT FireBeforeNavigate3 (IHTMLWindow2 *, IDispatch *, PCWSTR, DWORD, PCWSTR, PBYTE, PCWSTR, BOOL, BOOL *);

Availability

The ITridentService3 interface is implemented in

  • SHDOCVW version 6.0 from Internet Explorer 7, and thus also Windows XP SP3 and Windows Server 2003 SP2, but before Windows Vista;
  • IEFRAME version 7.0 and higher.

This interface is not documented by Microsoft in the January 2007 edition of the Windows Software Development Kit (SDK).

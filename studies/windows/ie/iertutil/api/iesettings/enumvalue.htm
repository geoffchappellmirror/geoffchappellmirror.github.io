EnumValue

Enumerates the values in an Internet Explorer setting that is a registry key.

Declaration 

HRESULT 
EnumValue (
    REFGUID Guid, 
    CEnum *Enum,);

in which CEnum is a class in the SettingStore namespace.

Parameters

The Guid argument specifies the setting to work on.

The Enum argument is the address of a class into which the function enters a list for the caller to enumerate.

Return Value

The function returns zero for success, else an error code.

Availability

The EnumValue function is exported from IERTUTIL as ordinal 681 in version 8.0 and higher.

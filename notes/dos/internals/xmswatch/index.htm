An Extended Memory Monitor

Chapter 15 of DOS Internals presents a program named XMSWATCH.EXE that draws on the work of many earlier chapters to report on extended memory usage. The program can be installed to stay resident, including to be loaded as a DOS device driver, so that reports include activity that has been monitored since installation but which would otherwise not be discoverable.

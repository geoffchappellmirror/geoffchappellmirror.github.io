CL Command-Line Error D2038

Message Text

invalid argument 'argument'

Circumstances

A command-line option that either allows or requires an argument was given with the cited argument, which is too long. The option itself is not cited. The present limit for the length of an argument in a command-line option is 1024 characters.

Documentation Status

The product documentation seems not to mention this error, perhaps thinking that its explanation is self-evident from the message text.

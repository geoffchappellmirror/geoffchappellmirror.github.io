Windows Options

Curiously, the MSHTML options include a handful whose ordinary location in the registry is explicitly not a matter for Internet Explorer:

  • HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion.

For no particular reason, these are here arranged alphabetically by value, regardless of the subkey. In recent Windows versions, one is also supported in policy keys. For interpretation, refer to the general discussion.

Key HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies
Value Allow Programmatic Cut_Copy_Paste
Type boolean
Default false
Availability MSHTML.DLL version 5.0 and higher

The Allow Programmatic Cut_Copy_Paste option has no known user-interface support.

Key HKEY_LOCAL_MACHINE\Software\Policies\Microsoft\Windows\CurrentVersion\Internet Settings
HKEY_CURRENT_USER\Software\Policies\Microsoft\Windows\CurrentVersion\Internet Settings
HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Internet Settings
Value DisableCachingOfSSLPages
Type boolean
Default false
Availability option: MSHTML.DLL version 5.50 and higher
policy: MSHTML.DLL version 6.0 from Windows Server 2003 SP1, and higher

The DisableCachingOfSSLPages option has user-interface support through Internet Properties in the Control Panel, as “Do not save encrypted pages to disk” in the Security group on the Advanced tab. It has user-interface support as a policy, too, through the Group Policy Object Editor, with the same description, in the Advanced Page category (itself under Internet Control Panel) of the Internet Explorer administrative template (displayed under Windows Components).

Key HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\ActiveDesktop
Value NoChangingWallpaper
Type dword
Default zero
Availability MSHTML.DLL version 5.0 and higher

The NoChangingWallpaper option has user-interface support as “Prevent changing wallpaper” in the Group Policy Object Editor, through the “Display” administrative template (displayed under “Control Panel”).

This same registry value is meaningful to SHELL32 version 4.71 and higher (though as NoChangingWallPaper).

EXPLORER Classes

The version 6.00 of EXPLORER.EXE from Windows Vista is a local server for the following creatable COM classes:

  • DesktopExplorerFactory
  • SeparateExplorerFactory
  • TaskBand
  • TrayBandSiteService
  • TrayDeskBand
  • TrayNotify

None are formally documented, but two are declared in SHOBJIDL.IDL (and thus have their CLSIDs defined in SHOBJIDL.H).

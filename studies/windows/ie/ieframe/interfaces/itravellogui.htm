ITravelLogUI

This interface is implemented in IEFRAME. It seems new to Windows Vista, though closer inspection may reveal it or something similar in some other module in an earlier Windows version. The IID is:

ITravelLogUI {92549FB6-2504-4018-83C5-0A950DF000F2}

This interface is not completely undocumented. It doesn’t get even a declaration in any Windows SDK header, but the corresponding IID_ITravelLogUI is defined in the UUID.LIB library.

Methods

The ITravelLogUI methods are:

Offset Method
0x0C Reset
0x10 AddTravelLogItem
0x14 ShowTravelLog

Prototypes

The following are known from Microsoft’s symbol files: 

HRESULT Reset (VOID);
HRESULT AddTravelLogItem (LONG, PCWSTR, ULONG);
HRESULT ShowTravelLog (INT, INT, HWND, RECT const *, LONG *);

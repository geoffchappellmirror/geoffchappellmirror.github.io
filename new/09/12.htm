New or Updated in December 2009

Although I do not intend to publish new material on Windows until I find full-time funding for the work, the recent arrival of Windows 7 on an MSDN disc has prompted me to start updating pages (mostly lists) for my own reference. It would be churlish to change these in my own copies rather than update them in the free resource. It helps that there turns out to be not nearly as much updating to do as might be expected: for all that Windows 7 is said to be so great a change of user experience from Windows Vista, the shell (in particular) has very few new functions, though there are a lot of new configurable settings.

If you get anything of value from this updating, then please support this site. Doing so may even encourage me to write up some real research when I’m done with this bookkeeping.

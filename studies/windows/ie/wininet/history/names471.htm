WININET 4.71 Named Exports

Version 4.71 of WININET.DLL is known only from Internet Explorer 4.0. Two additions to the API do not survive to version 5.0 and seem never to have been documented. Another might have been discontinued except that it had been documented: it is retained but as obsolete. Though many of the additions were documented immediately, a large proportion never have been and perhaps never will be.

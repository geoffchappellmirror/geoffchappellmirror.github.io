API-MS-Win-Core-DateTime-L1-1-0

All functions in the API-MS-Win-Core-DateTime-L1-1-0 set are documented exports from KERNEL32 in version 3.51 and higher:

  • GetDateFormatA
  • GetDateFormatW
  • GetTimeFormatA
  • GetTimeFormatW

The Windows 7 schema redirects this API Set to KERNEL32.

The Windows 8 schema redirects this API Set to KERNELBASE.

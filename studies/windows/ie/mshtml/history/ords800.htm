MSHTML 8.0 Ordinal-Only Exports

For version 8.0, MSHTML breaks a long-standing practice of exporting no functions by ordinal only. There are now five. None seem to be documented yet.

Ordinal Function
100 CreatePrivacIEDatabase
101 CreateUserFilter
102 CreateWebFilterCollection
103 GetPrivacIEUrlExpression
104 CreatePrivacIEAdvisor

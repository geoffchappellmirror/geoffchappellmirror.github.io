SYSTEM_VERIFIER_COUNTERS_INFORMATION

The SYSTEM_VERIFIER_COUNTERS_INFORMATION structure is what a successful call to ZwQuerySystemInformation or NtQuerySystemInformation produces in its output buffer when given the information class SystemVerifierCountersInformation (0x76).

Documentation Status

The SYSTEM_VERIFIER_COUNTERS_INFORMATION structure is not documented.

Layout

The SYSTEM_VERIFIER_COUNTERS_INFORMATION is 0xBC or 0x0100 bytes in 32-bit and 64-bit Windows, respectively.

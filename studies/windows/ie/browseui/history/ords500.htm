BROWSEUI 5.0 Ordinal-Only Exports

All but a handful of functions exported from BROWSEUI, right from the very first version, are exported only by ordinal. None are documented.

One of these functions did not survive even to version 5.50. Several were retired in Windows Vista: they continue to be exported, but only as forwards to SHUNIMPL.DLL for certain failure. Version 6.1 retired the remainder.

102 SHOpenFolderWindow retired in 6.1

107 SHExplorerParseCmdLine retired in 6.0 from Windows Vista

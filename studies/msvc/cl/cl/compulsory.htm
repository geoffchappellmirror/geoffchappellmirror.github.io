Compulsory Options

In the absence of any of the /LD, /LDd, /MD, /MDd, /ML, /MLd, /MT or /MTd options on the user-supplied command line, CL version 13.00.9466 enforces as a final contribution to the command line the following options:

  • -MT, if /clr is enabled (with or without its noAssembly keyword)
  • else -ML

Note that this late addition of /MT by default defines the _MT macro even if a /U_MT or /u on the user-supplied command line might otherwise have been intended as ensuring that the macro not be defined.

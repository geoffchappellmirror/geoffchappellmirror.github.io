Utility Functions

From very early on, SHELL32 has been a repository of miscellaneous utility functions, some documented, most not. The following are treated here as a set only because they all come to SHELL32.DLL from one object file (UTIL.OBJ). They date from version 4.00 unless otherwise stated. (One is from even earlier.)

MSHTML Classes: CMarquee

An instance of the CMarquee class is created for each MARQUEE tag.

Identifiers

coclass: HTMLMarqueeElement {3050F2B9-98B5-11CF-BB82-00AA00BDCE0B}
dispinterface: DispHTMLMarqueeElement {3050F527-98B5-11CF-BB82-00AA00BDCE0B}

Scripting

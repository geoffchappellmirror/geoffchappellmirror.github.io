LINK /NODEFAULTLIB

This option directs LINK to suppress either all default libraries or a specific default library.

Syntax

A shortening to /nod is permitted.

/nod[efaultlib][:library]

The optional library argument consists of everything after the colon.

Behaviour

With no argument, /nodefaultlib suppresses all default libraries. Subsequent use of /nodefaultlib with or without an argument is redundant, and is ignored without complaint.

After a /verbose with no argument, the first use of /nodefaultlib without an argument is reported to standard output, as is each use with an argument before any use without an argument.

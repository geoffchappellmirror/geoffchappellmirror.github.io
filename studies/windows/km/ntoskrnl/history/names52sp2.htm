Kernel Exports Added for Windows Server 2003 SP2

This page lists the functions that are first exported by the Windows kernel in the version 5.2 from Windows Server 2003 SP2. There are just two. Both are still exported as of version 6.1. Neither is documented in the Windows Driver Kit (WDK) for Windows Vista, dated January 2007, or declared in any WDK header file, but one has got documented since.

Function Remarks
KiCheckForSListAddress x86 only
MmDoesFileHaveUserWritableReferences undocumented until 2007;
documentation requires Windows Vista and higher;
declaration requires Windows Vista and higher

The x64 build of the kernel from Windows Server 2003 SP2 stops exporting KeServiceDescriptorTable.

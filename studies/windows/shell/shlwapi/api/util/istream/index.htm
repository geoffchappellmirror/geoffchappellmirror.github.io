IStream Functions

Starting with version 5.00, SHLWAPI provides a few utility functions to standardise work on objects that expose an IStream interface:

  • IStream_Read
  • IStream_ReadPidl (6.00)
  • IStream_Reset
  • IStream_Size
  • IStream_Write
  • IStream_WritePidl (6.00)
  • SHIsEmptyStream

None of these functions were yet documented by Microsoft as late as the January 2004 edition of the MSDN Library on CD. Most have got documented since, but sporadically. See for instance that a batch of documentation in 2004 somehow managed to include IStream_Read but miss IStream_Write.

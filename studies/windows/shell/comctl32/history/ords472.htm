COMCTL32 4.72 Ordinal-Only Exports

The following four functions date from Internet Explorer 4.0 and Windows 98 but were exported only by ordinal until the release of Windows XP. They are all documented, but as requiring Windows 2000 or even Windows XP.

Ordinal Function Remarks
410 SetWindowSubclass exported also by name in 5.82 from Windows XP SP1, and higher;
documented as requiring “version 5.8 or later”
411 GetWindowSubclass exported also by name in 6.0 and higher;
documented as requiring “version 6.0 or later”
412 RemoveWindowSubclass exported also by name in 5.82 from Windows XP SP1, and higher;
documented as requiring “version 5.8 or later”
413 DefSubclassProc exported also by name in 5.82 from Windows XP SP1, and higher;
documented as requiring “version 5.8 or later”

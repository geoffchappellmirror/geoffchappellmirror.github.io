BROWSEUI 6.0 Ordinal-Only Exports

Although version 6.0 added no functions that are exported by name, it did add several that are exported only by ordinal—and it didn’t add these all at once. None are documented. They continue to be exported in version 6.1, but only as forwards to SHUNIMPL.DLL for certain failure. Two are reimplemented in SHELL32.

Ordinal Function Remarks
118 UEMRegisterNotify retired in 6.1
136 SHEnumClassesOfCategories begins in Windows XP SP2 and Windows Server 2003 SP1;
retired in 6.1;
also ordinal 881 in SHELL32 6.1 and higher
137 SHWriteClassesOfCategories begins in Windows XP SP2 and Windows Server 2003 SP1;
retired in 6.1;
also ordinal 882 in SHELL32 6.1 and higher
138 SHStartDDEThread begins in Windows XP SP2 and Windows Server 2003 SP1;
retired in 6.1
139 SHCreateBrowserProgressAggregator begins in Windows Vista;
retired in 6.1

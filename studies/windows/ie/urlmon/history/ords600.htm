URLMON 6.0 Ordinal-Only Exports

Not until version 6.0 does URLMON start exporting functions only by ordinal. Unsurprisingly, the added function is not documented.

Ordinal Function Remarks
100 ZoneMappingToRegKey begins in Windows XP SP2 and Windows Server 2003 SP1

MSHTML Classes: CFrameElement

An instance of the CFrameElement class is created for each FRAME tag.

Identifiers

coclass: HTMLFrameElement {3050F314-98B5-11CF-BB82-00AA00BDCE0B}
dispinterface: DispHTMLFrameElement {3050F513-98B5-11CF-BB82-00AA00BDCE0B}

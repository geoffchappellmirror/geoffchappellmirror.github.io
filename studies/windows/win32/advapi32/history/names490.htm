Exports Added For ADVAPI32 4.90

Version 4.90 of ADVAPI32, which is known only for Windows Me, introduced one new function. It is exported from no other known version of ADVAPI32. Perhaps it was moved to another DLL. Microsoft eventually documented it, saying straight away that it is “not implemented and cannot be used”.

Function Remarks
CryptGetLocalKeyLimits discontinued in 5.0;
undocumented until 2004-2006;
documented as “not implemented” and requiring at least Windows Vista

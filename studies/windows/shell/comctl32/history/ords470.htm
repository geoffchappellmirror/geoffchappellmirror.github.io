COMCTL32 4.70 Ordinal-Only Exports

Version 4.70 rounds out the collection of string utility functions that date from at least version 3.50. The new ones for version COMCTL32 4.70 were duplicated contemporaneously (or perhaps not much later during development) into SHLWAPI, which is where they were eventually documented.

Ordinal Function Remarks
369 StrRChrIW retired in 5.82 and 6.10 from Windows Vista, and higher;
independent implementation exported from SHLWAPI version 4.70 and higher;
documented as requiring “shlwapi.dll version 4.71 or later”
376 IntlStrEqWorkerA retired in 5.82 and 6.10 from Windows Vista, and higher;
independent implementation exported from SHLWAPI version 4.70 and higher;
documented as requiring “shlwapi.dll version 5.0 or later”
377 IntlStrEqWorkerW retired in 5.82 and 6.10 from Windows Vista, and higher;
independent implementation exported from SHLWAPI version 4.70 and higher;
documented as requiring “shlwapi.dll version 5.0 or later”

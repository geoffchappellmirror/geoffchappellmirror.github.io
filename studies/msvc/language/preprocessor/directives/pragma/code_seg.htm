The “code_seg” Pragma

Syntax

#pragma code_seg ( [ operation [, label [, segment [, class ]]]] )

#pragma code_seg ( [ operation [, segment [, class ]]] )

#pragma code_seg ( [ segment [, class ]] )

where

  • operation is push or pop
  • label is an identifier
  • segment and class are strings

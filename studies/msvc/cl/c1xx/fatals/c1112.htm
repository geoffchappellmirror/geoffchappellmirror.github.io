Visual C++ Compiler Fatal Error C1112

Message Text 

compiler limit : 'number' too many macro arguments, only 'max' allowed

Circumstances

An attempt has been made to define a macro with more arguments than are supported. The cited number tells how many arguments were wanted (though the text, if interpreted literally, would have it that this is the number in excess of the maximum). Version 13.00.9466 can tolerate at most 127, which is shown as the max placeholder.

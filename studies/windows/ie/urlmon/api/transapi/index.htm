URLMON Translation Functions

The functions listed here are a miscellaneous collection that nowadays come to URLMON through one object file, named TRANSAPI.OBJ, but whose relationships with one another remain to be investigated.

Four of these functions date back to Internet Explorer 3.01 and two of these were even documented as early as the MSDN Library dated January 1997—but as COM functions with no mention of Internet Explorer. Nine more were added for Internet Explorer 4.0 and seem to have been documented promptly. And there the set was left for a long time, except for documenting GetClassFileOrMime some time during 1999 and 2000. The next addition was for late builds of version 6.0, starting from Windows XP SP2 and Windows Server 2003 SP1, with more for version 7.0. In the list below, functions are highlighted if they seem not to be documented in the Windows SDK dated January 2007.

  • CoInternetCanonicalizeIUri (7.0)
  • CoInternetCombineIUri (7.0)
  • CoInternetCombineUrl (4.71)
  • CoInternetCombineUrlEx (7.0)
  • CoInternetCompareUrl (4.71)
  • CoInternetCreateSecurityManager (4.71)
  • CoInternetCreateZoneManager (4.71)
  • CoInternetGetSecurityUrl (4.71)
  • CoInternetGetSecurityUrlEx (7.0)
  • CoInternetIsFeatureEnabled (6.0 from Windows XP SP2)
  • CoInternetIsFeatureEnabledForIUri (7.0)
  • CoInternetIsFeatureEnabledForUrl (6.0 from Windows XP SP2 and Windows Server 2003 SP1)
  • CoInternetIsFeatureZoneElevationEnabled (6.0 from Windows XP SP2 and Windows Server 2003 SP1)
  • CoInternetParseIUri (7.0)
  • CoInternetParseUrl (4.71)
  • CoInternetQueryInfo (4.71)
  • CoInternetSetFeatureEnabled (6.0 from Windows XP SP2 and Windows Server 2003 SP1)
  • FindMimeFromData (4.71)
  • GetClassFileOrMime (4.70)
  • IsValidURL (4.70)
  • URLDownload (4.70)
  • UrlMkGetSessionOption (4.71)
  • UrlMkSetSessionOption (4.70)

DUMPBIN /SYMBOLS

This option asks to dump the COFF Symbol Table.

Syntax

No arguments are anticipated. The option is /symbols only.

Behaviour

This option applies to executables in PE format, to COFF objects (whether as self-standing files or embedded in a library), and to debug files.

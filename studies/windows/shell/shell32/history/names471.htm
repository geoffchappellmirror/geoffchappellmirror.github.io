SHELL32 4.71 Named Exports

Of the functions that were introduced with SHELL32 version 4.71 (for Internet Explorer 4.0) and are exported by name, all are documented and all are said correctly to be available in versions 4.71 and higher (though DllInstall is documented as if implemented in SHLWAPI and SHGetDiskFreeSpace is documented as if it has both ANSI and Unicode forms).

  • DllCanUnloadNow
  • DllGetVersion
  • DllInstall
  • SHEmptyRecycleBinA
  • SHEmptyRecycleBinW
  • SHGetDiskFreeSpaceA
  • SHGetSettings
  • SHGetSpecialFolderPathA
  • SHInvokePrinterCommandA
  • SHInvokePrinterCommandW
  • SHQueryRecycleBinA
  • SHQueryRecycleBinW

Functions Previously Exported Only By Ordinal

One of the named exports has an older Unicode partner that was exported only by ordinal. Both become exported by name starting with version 4.71.

Function Ordinal First Availability
SHGetSpecialFolderPathW 175 4.0

Dynamic Arrays

From the very start, with version 4.00, COMCTL32 has provided a set of functions for working with collections of similar data without having to trouble over memory management and storage details. COMCTL32.DLL acquires them all from the one object file (DA.OBJ), but they are here approached in three categories:

  • Dynamic Pointer Arrays
  • Dynamic Structure Arrays
  • String Storage

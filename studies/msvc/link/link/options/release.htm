LINK /RELEASE

Syntax

No arguments are anticipated. The option is /release only.

The /release option is for the command line only. It is not valid as a directive in an object file.

Behaviour

The /release option is implied by the following /subsystem keywords: efi_application, efi_boot_service_driver, efi_rom, efi_runtime_driver and native.

DUMPBIN /UNWINDINFO

Syntax

No arguments are anticipated. The option is /unwindinfo only.

Behaviour

The /unwindinfo option implies /pdata. Its extra effect applies only to executables in PE format, and only then if targetted specifically to the IA64 machine.

The extra effect is that in each dump that /pdata enables of a function table, each entry is followed by a dump of unwind information (if present). As with the function table, the unwind information is part of the executable’s image and is expected to lie wholly within some section of that image. However, the unwind information is not necessarily in the same section as the function table. It is dumped even if the section that contains it is not eligible for dumping (as determined by /range and /section options).

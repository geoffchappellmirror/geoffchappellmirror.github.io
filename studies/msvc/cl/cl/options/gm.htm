CL /GM

Syntax

The option is /GM only.

Relations

This option has no syntactic relations with other options: it does not override any; it is not incompatible with any; it has no prerequisites.

Indeed, this option appears to be reserved. It is syntactically valid, such that its use is not an error, but it has no effect.

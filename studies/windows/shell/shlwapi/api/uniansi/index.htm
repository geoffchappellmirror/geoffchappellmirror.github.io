Character Set Conversions

From version 5.00, SHLWAPI provides some functions to ease the work of converting strings between Unicode and ANSI character sets:

  • SHAnsiToAnsi
  • SHAnsiToUnicode
  • SHAnsiToUnicodeCP
  • SHUnicodeToAnsi
  • SHUnicodeToAnsiCP
  • SHUnicodeToUnicode

These functions are exported by ordinal only. None were documented as of January 2004, though four (without the CP suffix) were documented sometime later in 2004.

URLMON 4.70 Named Exports

Version 4.70, released with Windows 95 OSR2 and Internet Explorer 3.01 and 3.02, more than doubled the set of exported functions from the original Internet Explorer 3.00. Almost all the new functions are documented, though with “Internet Explorer 3.0” given as the minimum availability, which overlooks that these functions actually aren’t exported from the original.

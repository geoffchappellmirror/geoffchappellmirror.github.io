MSHTML Classes: CPeerUrnCollection

The CPeerUrnCollection class is the internal implementation of what Microsoft’s HTML and DHTML Reference refers to as the behaviorUrns collection. Instances are created by reading the behaviorUrns property of an element.

Identifiers

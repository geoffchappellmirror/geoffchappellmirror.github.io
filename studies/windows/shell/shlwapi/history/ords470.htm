SHLWAPI 4.70 Ordinal-Only Exports

Only one function is exported without a name by SHLWAPI version 4.70. Roughly eight years later, it got documented.

Ordinal Function Remarks
1 ParseURLA exported also by name in 6.0 from Windows Vista, and higher;
documented as requiring “version 5.0 or later”
2 ParseURLW exported also by name in 6.0 from Windows Vista, and higher;
documented as requiring “version 5.0 or later”

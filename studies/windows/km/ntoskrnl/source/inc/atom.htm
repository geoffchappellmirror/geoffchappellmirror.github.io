ATOM.H

The public symbol file NTKRPAMP.PDB for the original release of Windows 10 tells that the kernel is built with the ATOM.H header at

d:\th.public.fre\internal\minwin\priv_sdk\inc

and draws from it the following type definitions:

Line Number Type
26 struct _RTL_ATOM_TABLE_REFERENCE
35 struct _RTL_ATOM_TABLE_ENTRY
44 struct _RTL_ATOM_TABLE

The header ATOM.H is not known in any Device Driver Kit (DDK) or Windows Driver Kit (WDK).

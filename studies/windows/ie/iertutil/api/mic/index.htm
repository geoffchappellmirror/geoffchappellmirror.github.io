Mandatory Integrity Control

IERTUTIL version 7.0 exports a dozen functions to help with Mandatory Integrity Control (MIC), and version 8.0 does not add to them:

  • GetLowIntegrityToken
  • IEStubIsLuaEnabled
  • IsBrowserProcess
  • IsElevatedProcess
  • IsProtectedModeCOMCaller
  • IsProtectedModeEnabled
  • IsProtectedModeEnabledForIE
  • IsProtectedModeProcess
  • LaunchIEInProtectedMode
  • RegGetDwordValue
  • Shim_CoCreateInstance
  • Shim_CoGetClassObject

Note that a few of these seem to be helpers which must in turn have seemed useful enough to export.

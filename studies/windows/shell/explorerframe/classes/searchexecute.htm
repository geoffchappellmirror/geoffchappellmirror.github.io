SearchExecute

The SearchExecute class is exposed by ExplorerFrame.dll through its DllGetClassObject function. Instances can therefore be created, using such well-documented COM functions as CoCreateInstance, by anyone who knows the CLSID:

CLSID_SearchExecute {90B9BCE2-B6DB-4FD3-8451-35917EA1081B}

The class implements the following interfaces:

  • IExecuteCommand
  • IObjectWithAssociationElement
  • IObjectWithSelection
  • IObjectWithSite

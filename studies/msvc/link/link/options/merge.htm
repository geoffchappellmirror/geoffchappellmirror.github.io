LINK /MERGE

Syntax

/merge:from=to

The required argument consists of everything after the colon. It is taken in two parts, from and to, separated by an equals sign. It is a fatal error (LNK1117) if there is no equals sign or if either from or to is empty. It is a fatal error (LNK1184) if either from or to contains either a dollar sign or space.

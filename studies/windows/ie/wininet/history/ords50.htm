WININET 5.0 Ordinal-Only Exports

Not until version 5.0 does WININET export any functions only by ordinal. None are yet documented. Two are just ANSI and Unicode forms of a function that is exported by name without a suffix.

Ordinal Function
101 DoConnectoidsExist
102 GetDiskInfoA
103 PerformOperationOverUrlCacheA
104 HttpCheckDavComplianceA
105 HttpCheckDavComplianceW
108 ImportCookieFileA
109 ExportCookieFileA
110 ImportCookieFileW
111 ExportCookieFileW
112 IsProfilesEnabled

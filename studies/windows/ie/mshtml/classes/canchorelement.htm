MSHTML Classes: CAnchorElement

An instance of the CAnchorElement class is created for each A tag.

Identifiers

coclass: HTMLAnchorElement {3050F248-98B5-11CF-BB82-00AA00BDCE0B}
dispinterface: DispHTMLAnchorElement {3050F502-98B5-11CF-BB82-00AA00BDCE0B}

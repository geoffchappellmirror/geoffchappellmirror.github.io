SYSTEM_NUMA_PROXIMITY_MAP

The SYSTEM_NUMA_PROXIMITY_MAP structure provides input to and receives output from the ZwQuerySystemInformation or NtQuerySystemInformation functions when given the information class SystemNumaProximityNodeInformation (0x65).

Documentation Status

The SYSTEM_NUMA_PROXIMITY_MAP structure is not documented.

Layout

The SYSTEM_NUMA_PROXIMITY_MAP is 0x08 bytes in both 32-bit and 64-bit Windows.

Offset Definition Remarks
0x00 
ULONG NodeProximityId;
 input
0x04 
USHORT NodeNumber;
 output

Credits for Internet Mail and News

Run either Internet Mail or Internet News. From the Help menu, select About Microsoft Internet Mail and News. This produces a dialog box inside of which is a list box and an OK button. To start with, the OK button has the focus. Click on the list box, then type “mortimer”. (When typing the expected keys, you may insert extra keystrokes or hold down the Ctrl, Alt or Shift keys without affecting the result.) The dialog box changes to show a multi-coloured scrolling list of credits.
